## 我能做什么

<img src="./assets/capability-grid.svg" alt="能力地图：Agent、RAG、Backend、GIS" width="100%" />

- **Agent 工作流**：任务拆解、工具调用、状态流转、事件推送和可观察执行过程。
- **RAG 系统**：全文检索、向量检索、结果融合、重排策略和领域知识库组织。
- **后端与自动化**：Python 服务、小型 CLI、浏览器脚本、协议适配和日常流程工具化。
- **GIS / Web 可视化**：空间数据、地图交互、WebGL、PostGIS 和前端可视化表达。
