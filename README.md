# GeoSyntax

专注 Python、Agent、RAG 与 GIS/Web 可视化。  
把检索、自动化、协议适配和空间数据工具做成能长期维护的小项目。

## 正在关注

- Agent 工作流：任务拆解、工具调用、流式状态与可观察执行过程。
- RAG 系统：全文检索、向量检索、重排、领域文档整理。
- Web 与自动化：TypeScript、Python、Go、小型 CLI 和浏览器侧工具。
- GIS / Visualization：空间数据、Web GIS、地图交互和可视化工程。
